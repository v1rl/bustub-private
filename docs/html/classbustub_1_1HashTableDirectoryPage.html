Detailed Description

Directory Page for extendible hash table.

Directory format (size in byte):

| LSN (4) | PageId(4) | GlobalDepth(4) | LocalDepths(512) | BucketPageIds(2048) | Free(1524)

◆ GetGlobalDepthMask()

auto bustub::HashTableDirectoryPage::GetGlobalDepthMask ( ) -> uint32_t

GetGlobalDepthMask - returns a mask of global_depth 1's and the rest 0's.

In Extendible Hashing we map a key to a directory index using the following hash + mask function.

DirectoryIndex = Hash(key) & GLOBAL_DEPTH_MASK

where GLOBAL_DEPTH_MASK is a mask with exactly GLOBAL_DEPTH 1's from LSB upwards. For example, global depth 3 corresponds to 0x00000007 in a 32-bit representation.

Returns
mask of global_depth 1's and the rest 0's (with 1's from LSB upwards)

void bustub::HashTableDirectoryPage::VerifyIntegrity ( )

VerifyIntegrity

Verify the following invariants: (1) All LD <= GD. (2) Each bucket has precisely 2^(GD - LD) pointers pointing to it. (3) The LD is the same at each index with the same bucket_page_id
