Detailed Description

Slotted page format:

| HEADER | ... FREE SPACE ... | ... INSERTED TUPLES ... |

^ free space pointer

Header format (size in bytes):

| NextPageId (4)| NumTuples(2) | NumDeletedTuples(2) |

| Tuple_1 offset+size (4) | Tuple_2 offset+size (4) | ... |

Tuple format: | meta | data |
