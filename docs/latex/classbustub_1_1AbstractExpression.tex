\doxysubsection{Detailed Description}
\mbox{\hyperlink{classbustub_1_1AbstractExpression}{Abstract\+Expression}} is the base class of all the expressions in the system. Expressions are modeled as trees, i.\+e. every expression may have a variable number of children.
