A list of forward links. 

Note\+: {\ttfamily links\+\_\+\mbox{[}0\mbox{]}} is the lowest level link, and {\ttfamily links\+\_\+\mbox{[}links\+\_\+.\+size()-\/1\mbox{]}} is the highest level link.

We use {\ttfamily std\+::shared\+\_\+ptr} to manage the memory of the next node. This is because the next node can be shared by multiple links. We also use {\ttfamily std\+::vector} instead of a flexible array member for simplicity instead of performance.
