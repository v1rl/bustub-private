\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classbustub_1_1AbstractExecutor}{Abstract\+Executor}} implements the Volcano tuple-\/at-\/a-\/time iterator model. This is the base class from which all executors in the Bust\+Tub execution engine inherit, and defines the minimal interface that all executors support.
