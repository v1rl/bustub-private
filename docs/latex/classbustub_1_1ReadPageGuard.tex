\doxysubsection{Detailed Description}
An RAII object that grants thread-\/safe read access to a page of data. 

The {\itshape only} way that the Bus\+Tub system should interact with the buffer pool\textquotesingle{}s page data is via page guards. Since {\ttfamily \mbox{\hyperlink{classbustub_1_1ReadPageGuard}{Read\+Page\+Guard}}} is an RAII object, the system never has to manually lock and unlock a page\textquotesingle{}s latch.

With {\ttfamily \mbox{\hyperlink{classbustub_1_1ReadPageGuard}{Read\+Page\+Guard}}}s, there can be multiple threads that share read access to a page\textquotesingle{}s data. However, the existence of any {\ttfamily \mbox{\hyperlink{classbustub_1_1ReadPageGuard}{Read\+Page\+Guard}}} on a page implies that no thread can be mutating the page\textquotesingle{}s data.

Note that we do not EVER want use a guard that has only been default constructed. The only reason we even define this default constructor is to enable placing an \char`\"{}uninitialized\char`\"{} guard on the stack that we can later move assign via {\ttfamily =}.
