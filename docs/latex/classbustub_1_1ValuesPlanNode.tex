\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classbustub_1_1ValuesPlanNode}{Values\+Plan\+Node}} represents rows of values. For example, {\ttfamily INSERT INTO table VALUES ((0, 1), (1, 2))}, where we will have {\ttfamily (0, 1)} and {\ttfamily (1, 2)} as the output of this executor.
