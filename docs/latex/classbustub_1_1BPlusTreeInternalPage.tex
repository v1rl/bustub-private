\doxysubsection{Detailed Description}
Store {\ttfamily n} indexed keys and {\ttfamily n + 1} child pointers (page\+\_\+id) within internal page. Pointer PAGE\+\_\+\+ID(i) points to a subtree in which all keys K satisfy\+: K(i) $<$= K $<$ K(i+1). NOTE\+: Since the number of keys does not equal to number of child pointers, the first key in key\+\_\+array\+\_\+ always remains invalid.
