\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classbustub_1_1DeletePlanNode}{Delete\+Plan\+Node}} identifies a table from which tuples should be deleted. The tuple(s) to be updated come from the child of the \mbox{\hyperlink{classbustub_1_1DeleteExecutor}{Delete\+Executor}}.

NOTE\+: To simplify the assignment, \mbox{\hyperlink{classbustub_1_1DeletePlanNode}{Delete\+Plan\+Node}} has at most one child.
