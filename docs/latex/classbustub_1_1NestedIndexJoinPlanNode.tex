\doxysubsection{Detailed Description}
\mbox{\hyperlink{classbustub_1_1NestedIndexJoinPlanNode}{Nested\+Index\+Join\+Plan\+Node}} is used to represent performing a nested index join between two tables The outer table tuples are propagated using a child executor, but the inner table tuples should be obtained using the outer table tuples as well as the index from the catalog.
