\doxysubsection{Detailed Description}
\mbox{\hyperlink{classbustub_1_1AggregationPlanNode}{Aggregation\+Plan\+Node}} represents the various SQL aggregation functions. For example, COUNT(), SUM(), MIN() and MAX().

NOTE\+: To simplify this project, \mbox{\hyperlink{classbustub_1_1AggregationPlanNode}{Aggregation\+Plan\+Node}} must always have exactly one child.
