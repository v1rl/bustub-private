\doxysubsection{Detailed Description}
\mbox{\hyperlink{classbustub_1_1AbstractPlanNode}{Abstract\+Plan\+Node}} represents all the possible types of plan nodes in our system. Plan nodes are modeled as trees, so each plan node can have a variable number of children. Per the Volcano model, the plan node receives the tuples of its children. The ordering of the children may matter.
